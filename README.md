A modern, responsive **Sign-Up & Login web app UI** built using HTML, CSS, and JavaScript.

---

## 📌 About The Project

Basket™ Sign-Up Form is a frontend project that simulates a real-world authentication system UI. It includes both **sign-up and login pages**, form validation, and success feedback.

This project focuses on improving **frontend development skills, UI design, and JavaScript form handling**.

---

## 🎯 Features

- ✅ Modern UI design
- ✅ Fully responsive (Mobile + Desktop)
- ✅ Real-time form validation
- ✅ Password confirmation check
- ✅ Login navigation
- ✅ Success screen after signup
- ✅ Clean and structured code

---

## 🛠️ Built With

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
